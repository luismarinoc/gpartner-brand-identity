# Logo y Uso — GPartner Consulting

## Descripción

El logotipo de GPartner combina:

1. **Isotipo (símbolo):** una cabeza estilizada de **felino (jaguar/león)** construida a partir de una letra **"G"**, formada con trazos en navy, verde y rojo. Transmite fuerza, agilidad y precisión.
2. **Logotipo (wordmark):** "**GPARTNER**" en mayúsculas, bold, navy; bajo él "**CONSULTING**" en mayúsculas con tracking amplio.
3. **Separador vertical** tricolor (rojo / verde / navy).
4. **Descriptor:** "**TECHNOLOGY PARTNER**" en verde, a la derecha del separador.

## Versiones recomendadas

- **A color sobre fondo claro/blanco:** versión principal.
- **Blanco sobre fondo navy/oscuro:** versión monocromática blanca.
- **Isotipo solo (la "G" felina):** para favicon, avatares, app icons y espacios reducidos.

## Reglas de uso

**Hacer:**
- Mantener un área de protección (clear space) mínima equivalente a la altura de la "G" alrededor del logo.
- Usar siempre los colores oficiales (ver `tokens/colors.json`).
- Preservar las proporciones; escalar de forma uniforme.
- Sobre fondos con imagen, usar la versión blanca o una caja de color sólido.

**No hacer:**
- No deformar, rotar ni inclinar el logo.
- No cambiar los colores ni aplicar degradados ajenos a la marca.
- No reordenar o separar los elementos del wordmark.
- No colocar el logo a color sobre fondos de bajo contraste.
- No añadir sombras, contornos ni efectos.

## Favicon

La web usa un favicon derivado del isotipo (la "G" felina). Para productos digitales, exportar el isotipo en 512×512, 192×192, 48×48 y 32×32.

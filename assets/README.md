# Assets de logo — GPartner Consulting

## Logo oficial (vectorial, versionado en el repo)
**`gpartner-logo.svg`** — logo vectorial oficial entregado por el equipo. Escala sin pérdida, peso mínimo, renderiza en GitHub y en cualquier navegador. **Úsalo como fuente principal.**

En HTML/React:
```html
<img src="assets/gpartner-logo.svg" alt="GPartner Consulting">
```

## Versiones ráster (para alojar en el repo si se necesitan)
Binarios entregados en la carpeta local; arrástralos a esta carpeta en GitHub (web) o con `git add`:
- `gpartner-logo-transparent.png` — color, fondo transparente, 2000×352, sin pérdida.
- `gpartner-logo-transparent-1300.png` — igual, 1300×229 (más liviano).
- `gpartner-logo-color.png` — color, fondo blanco, 2000×352.
